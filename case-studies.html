Winner | November 2024

Network Management Platform for Healthcare Provider Optimization

Challenge: Built an AI-powered network management tool to help health insurers identify and recruit high-value providers using price transparency data and advanced analytics.

Key Solutions Delivered
  • Data Quality & Bias Reduction Engine
    • Designed a hierarchical swarm agent system using machine learning and NLP to automatically validate, standardize, and clean provider data from multiple sources
    • Implemented anomaly detection across three levels using ACO and PSO algorithms to identify pricing outliers and performance inconsistencies
    • Created counterfactual analysis tools enabling "what-if" recruitment scenarios to predict network performance outcomes
  • Performance Analytics Dashboard
    • Developed comprehensive provider scoring algorithms combining patient satisfaction, treatment success rates, readmission rates, and mortality data
    • Built real-time benchmarking tools with geospatial analysis to compare providers across health plans and regions
    • Integrated quality metrics from CAHPS surveys, CMS scores, and referral volume data for accurate provider assessment
  • Automated Recruitment System
    • Architected end-to-end automation using Apache Nifi, PostgreSQL, and UiPath RPA for streamlined provider outreach
    • Implemented ML-powered recommendation engine using K-means clustering to identify optimal provider recruitment targets
    • Created compliance monitoring with automated alerts for HIPAA and CMS regulatory requirements

Impact: Addressed critical healthcare challenges where 30% of provider data changes annually and only 50% of hospitals comply with price transparency rules, enabling insurers to build high-performing narrow networks through data-driven provider selection.

Technologies: Apache Kafka, PostgreSQL, Machine Learning (AIF360, Fairlearn), NLP, RPA (UiPath), PowerBI/Tableau
