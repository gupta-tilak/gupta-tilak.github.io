Chatty - Real-time Chat Application

Full Stack Development Project | May 2025 - June 2025

Chatty is a full-stack real-time chat application that enables instant messaging with a modern, responsive interface. The application features secure JWT authentication, live user status updates, and comprehensive profile management with Cloudinary image uploads. Users can customize their theme preferences while enjoying seamless real-time communication. The project demonstrates expertise in modern web development practices, including modular RESTful API design and real-time data synchronization.
